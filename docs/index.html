Built for real agents

RRF Fusion Recall

4 parallel ranked lists (SDR + MinHash + Tag Jaccard + optional embeddings) fused with Reciprocal Rank Fusion, k=60. Graph walk expands results via typed connections.

Living Memory

8-phase background maintenance: decay, reflection, insights, consolidation, cross-connections, archival, health check, persistence. No LLM needed for any phase.

4-Level Hierarchy

Working (0.80) → Decisions (0.90) → Domain (0.95) → Identity (0.99). Records promote on repeated access, demote through decay. Like human memory.

Trust & Provenance

Source authority scoring with automatic provenance stamping. Credibility tracking for 60+ domains. Trust-weighted recall ranking.

Auto-Protect Guards

Regex-based PII detection: phone numbers, emails, crypto wallets, API keys, passwords. Automatic protective tagging prevents data loss during consolidation.

Pluggable Embeddings

Works without embeddings. Optionally plug in any embedding function (OpenAI, Ollama, sentence-transformers) as a 4th RRF signal for semantic boost.

Typed Connections

Causal, reflective, associative, coactivation links between memories. 2-hop graph walk with damping. Causal chain following for reasoning context.

Research Orchestrator

Start research projects, collect findings with source credibility, synthesize results. Built-in domain credibility scoring for 60+ sources.

Circuit Breaker

Per-tool failure tracking with automatic open/close. Prevents cascading failures when external tools go down. Built-in cooldown and health reporting.

Use cases

Local AI Assistant

Aura + Ollama = fully local AI with persistent memory. No cloud, no API keys, no data leaves your machine.

pip install aura && ollama pull llama3.2

Edge / IoT / Embedded

2.7 MB binary with encryption. Smart home hubs, industrial sensors, medical devices, air-gapped systems.

Multi-Agent Systems

Each agent gets its own Aura brain. Trust scoring ensures agent-generated content is weighted differently from user input.

tc.source_trust = {"agent": 0.7, "user": 1.0}
